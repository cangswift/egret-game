# egret连连看小游戏

## 特色
- [x]   关卡定制，每个关卡要清除的元素和背景图都不一样
- [x]   即时通讯模块，私聊及群聊

## 主要功能
- [x]   解锁和选择关卡、元素
- [x]   游戏道具
- [x]   游戏商城
- [x]   游戏关卡
- [x]   存储
